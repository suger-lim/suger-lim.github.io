为什么写博客

发表于 2017-03-22 | 分类于 思考与讨论

我是一个混迹在互联网行业边缘的待业者，大半年的时间里，看过一些开发工程师的招聘要求，其中有一些会希望应聘者有Github的使用经验或者有自己的博客。作为加分项，这可能是促使我也想写博客的一个诱因。但是，我却不想承认这个事实？于是写一篇名为《为什么写博客》的文章，开启写作之旅。

巩固与分享知识

虽然已经工作了2年多，但是这两年的工作内容看起来是非常凌乱的。所以希望能通过分析总结工作学习中遇到的问题、知识点，形成统一的框架。从而达到提升知识储备，从容应对新问题的状态。另外，通过分享知识，可以加深学习效果。

具象的思考环境

不单单在技术上，我有着想提升自己的诉求，我更想做的是能创造一个具象的思考环境，抽象的大脑思考，经常让外界的环境影响思考后的行为活动。也就是说，我想要独立思考之后，能独立行动（这里按照自己的意愿行动）。

塑造个人形象

“我是xxx,我为自己代言。”我想通过一条清晰的心路历程，把真实的自己展现出来。目的是什么？寻求志同道合的伙伴。

总结

感觉自己写的第一篇博客有点过于简单，缺少的就是写作的能力吧。总之，是把自己写作目的写清楚了。接下来，就是持续性的写作了。
