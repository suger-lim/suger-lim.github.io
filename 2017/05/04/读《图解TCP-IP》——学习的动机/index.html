读《图解TCP/IP》——学习的动机

发表于 2017-05-04 | 分类于 思考与讨论

“现代计算机网络的首要目的之一，可以说是连接人与人。置身于世界各地的人们可以通过网络建立联系、互相沟通、交流思想。” ——《图解TCP/IP》第10页

那作为一个从事网络通信的技术人员的我，学习的首要目的是什么？动机是什么？
是枯燥地积累，然后获得出色的能力吗？是为了更效率地解决网络产品出现的问题吗？是让自己成为技术牛人，然后获得更多的报酬吗？还是仅仅是因为工作要挟，不然就要被淘汰？我不能否认上面的这些学习动机，如果是几年前没毕业的时候，如果是在和他人谈论时，如果是在缘分地读到这本书之前，如果是在我没有更高的追求的心态下，也许这些动机就是我的学习目的。不过，还好，这些条件都不成立，让我看清了职业与理想的关联——计算机网络的理想，不正是我应该有的理想吗？

王国维在《人间词话》开始就说：“词以境界为最上。有境界，则自成高格，自有名句。五代、北宋之词所以独绝者在此。” 所以我认为做技术、学知识的目的也不该仅仅着眼于技术本身，而是要有境界，这样也会改善自己看待问题的角度。这就让我不得不反思一下这些年来种种对待问题时的妥协态度。

追求优秀如果在环境看来格格不入时，能不能坚持自我，真的显得格外重要。就拿这本书的作者来说，是由4位日本技术专家写的，他们不仅能认识到计算机网络的目的，而且能在书中表达出来——“建立联系、互相沟通、交流思想”，这其实能从侧面反映出一种态度，就是服务意识。我们不论在社会中扮演什么样的角色，我们不论是人还是物，都不是为了纯粹的享受，而是把自己定义为一个合理的角色，为社会的更高效合理运转提供动力。这种态度就是一种好的东西，我们既然落后，就更应该摆正自己的态度，而不是，不但见贤不思齐，反而找出一堆特殊条件、历史原因继续停留在落后的层次。这可能不仅仅是落后了，简直就是一种“恐优症”，病灶是内心极度的不自信，现象是强词夺理。

“志当存高远”，小学毕业的时候，班主任给我留下的一句箴言，让我对出下一句。我一直在想，怎么工整地漂亮地写在纸上。直到现在，我才明白，下一句并不是5个字的句子，而是不论什么时候，都要心存鸿鹄之志地生活。
