Data Service

发表于 2017-04-29 | 分类于 工作

这篇博客的目的有两个，一是明确一下要接的offer，二是做好接offer的相关准备。

关于中科创达的Data Service Team

通过跟技术经理的交流以及下面的职位描述，比较明确了职位的职责主要是处理手机PS相关业务（如AT command,MMS,搜网等）问题。具体到工作内容就是和高通的技术支持岗位相同，通过高通的技术支持系统，接收由客户提出的case，分析问题，如果需要修改，交由研发部门部门给出解决方案，并配合研发部门与客户完成解决方案的实施。

Data Service 职位描述

1.Support Qualcomm multimode chipset customer for design-win projects, Help chipset OEMs to understand QCT modem software/solution, give customers training on QCT products.
2.Play expert role, trouble shooting, identify problem reported by customers, analysis and report issues to developer team, help developer team to narrow down the root cause and fix problems.
3.Responsibilities for Data related Embedded Application and Wireless protocol Software, focus on PS like AT command, MMS, network surfing.

对比中兴offer

中兴的5G测试岗位工作内容就是测试，接触问题本质的分析可能会比较少。相比下，中科创达的offer，能接触到比较多的高通资源，而且能跟高通工程师一起合作，我判断这个岗位的成长性以及挑战会更适合我。总结而言，在资源、环境上更适合我的成长。所以，我决定接这个offer。

准备

知识准备

1.TCP/IP协议的复习（《图解TCP/IP》）
2.Wireshark的学习（《Wireshark网络分析实战）
3.AT命令的复习
4.3GPP协议的准备

###工作规划
因为较长时间没有工作了，总结之前的工作经验、找工作的教训，应该对自己的工作态度，工作方法，学习方法提前做一下规划。

1.工作态度：在公司，工作是第一位的。上下班时间，手机，工作之外的书籍，闲聊等都是要注意的；对待问题的积极性、看待问题的角度、解决问题的毅力是最重要的。
2.工作方法：对待问题要追根问底；合理安排优先级；细节要三遍review，减少失误。
3.学习方法：在公司，要勇于请教，但是一定要提前做好功课，不能不会就问，对自己的提高不利；在家里要及时整理知识点，以备分享到博客。
